# Sådan bruger du værktøjet

1. Klon eller download repoet.
2. Åbn `index.html` i din browser.
3. Vælg klinge/tandhjul, hjul/dæk (hvis relevant i dit tool), og kadence.
4. Brug CSV-eksport hvis du vil sammenligne setups.

> Tip: Hvis du ændrer i koden, kan du bare reloade siden – ingen build kræves.
