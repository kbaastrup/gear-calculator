# FAQ

## Skal jeg installere noget?
Nej. Værktøjet kan køres lokalt ved at åbne `index.html`.

## Kan jeg bruge det på mobil?
Ja, hvis UI’et er responsivt – ellers fungerer det bedst på desktop.

## Hvor præcist er det?
Det er et referenceværktøj. Små afvigelser kan opstå pga. dækvolumen, tryk og reel omkreds.
